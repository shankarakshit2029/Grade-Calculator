# Grade Calculator

## Overview
The Grade Calculator application allows users to calculate their grades based on various assignments, tests, and exams. It provides an easy-to-use interface for inputting scores and generates average results and final grades based on user-defined criteria.

## Features
- **User-friendly Interface**: Easy to navigate layout for inputting grades.
- **Dynamic Calculations**: Automatically updates grades as users enter new scores.
- **Customizable Grade Criteria**: Users can set the weight of different assignments to reflect their class structure.
- **Final Grade Output**: Provides a final grade based on the input data, formatted as a letter grade if desired.
- **Save and Load Scores**: Ability to save user data and load it back to continue calculations.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/shankarakshit2029/Grade-Calculator.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Grade-Calculator
   ```
3. Install the required dependencies.
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python app.py
   ```

## Usage
- Open the Grade Calculator app. 
- Input your scores for each assignment as prompted.
- Adjust the weights for each category if necessary.
- Click on 'Calculate' to get your final grade.
